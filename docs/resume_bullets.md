# Resume Bullets

- Built a public baseball analytics research portfolio in Python focused on pitch design, swing decisions, and run value estimation using Statcast-compatible data workflows.
- Designed modular data ingestion, preprocessing, evaluation, and plotting utilities to support reproducible baseball modeling projects rather than one-off notebook analysis.
- Engineered pitch movement, velocity, count-state, and location features for player evaluation and strategy-oriented modeling tasks.
- Developed an unsupervised pitch-shape clustering workflow using standardized movement features, dimensionality reduction, and cluster-level pitch summaries.
- Built a baseline swing decision classifier using pitch location and movement features, with ROC AUC, log loss, and confusion-matrix-based evaluation.
- Created a context-aware run value modeling workflow connecting pitch-level outcomes to expected offensive value using regression baselines and grouped summaries.
- Wrote concise project reports translating technical methods, limitations, and next steps into baseball-facing research communication.
- Structured raw and processed public data pipelines to keep source assumptions explicit and analysis outputs reproducible.
- Applied Python scientific computing tools including pandas, NumPy, scikit-learn, matplotlib, SciPy, and XGBoost in an independent research setting.
- Used agentic coding tools to accelerate implementation while maintaining readable code, testing, and documentation standards.
