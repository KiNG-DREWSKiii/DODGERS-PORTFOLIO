# Resume Bullets

- Developed Python-based baseball analytics models using public Statcast-style data.
- Built modular workflows for data ingestion, feature engineering, clustering, and classification.
- Conducted exploratory modeling of pitch movement, swing decisions, and run value estimation.
- Created reusable utilities for preprocessing, plotting, and model evaluation.
- Wrote concise reports summarizing methods, outputs, and limitations.
- Applied pandas, NumPy, scikit-learn, matplotlib, and pybaseball in project-based research work.
