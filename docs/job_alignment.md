# Job Alignment

This repository is designed to align directly with a Junior Quantitative Analyst role in baseball operations.

- **Model development:** each project frames a baseball question, engineers features from public pitch-level data, and fits a baseline model that can be extended.
- **Clean code:** reusable utilities, modular project structure, and lightweight tests emphasize maintainability instead of notebook-only analysis.
- **Reproducible workflow:** raw and processed data directories, script entry points, and concise reports support a research process that can be rerun and reviewed.
- **Baseball research questions:** the project set covers pitch shape, swing decisions, and run value, which map naturally to player evaluation, development, and strategy.
- **Communication of findings:** each project includes a notebook and a written memo focused on methods, limitations, and next steps.

This framing is suitable for application materials because it shows curiosity, quantitative discipline, and the ability to translate open-ended baseball questions into structured analysis.
